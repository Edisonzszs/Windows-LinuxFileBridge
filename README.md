# Windows-Linux 文件桥

## 启动
- 双击 `start_gui.vbs`
- 或在当前目录执行：`python wsl_file_bridge_gui.py`

## 已实现
- Windows 和 Linux 目录都可随时选择
- 双向手动传输（文件和文件夹，多选批量）
- 快捷方式文件（`.lnk`、`.url`）默认不在列表中展示，也不会参与传输
- Office 临时文件（`~$` 开头）和 `desktop.ini` 默认隐藏
- 同名冲突自动重命名，保留两份
- 双侧删除所选，并在删除前弹窗二次确认
- 历史记录（含失败记录）
- 失败任务一键重试
- 清空历史时二次确认
- 文件类型筛选：全部文件、文件夹、常用文档、图片、压缩包、代码文件、媒体文件、数据文件、安装与镜像
- 具体类型筛选只显示匹配文件，不再混入文件夹
- 文件夹筛选只显示目录
- 列表图标：文件夹、压缩包、文档、图片、媒体、代码、安装文件、快捷方式
- 新建文件夹：顶部按钮 + 右键菜单
- 右键菜单：重命名、打开所在位置、查看属性、新建文件夹、删除所选

## 拖拽
- 已预留拖拽区
- 若未安装 `tkinterdnd2`，拖拽功能会提示未启用
- 安装后可启用系统拖拽：`pip install tkinterdnd2`

## 说明
- Linux 路径通过 `\\wsl$\发行版` 访问
- 默认发行版是 `Ubuntu-24.04`，可在界面顶部修改
- 右键操作建议一次选择 1 个项目，删除除外
